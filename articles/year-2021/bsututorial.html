The Bash Scripting Utility official Tutorial.

Downloading the utility:

  1. Install git by using your package manager. (e.g: pacman -S git)
  2. Run git clone https://github.com/0xBooper/BashScriptingUtility.git
  3. Cd into the directory where it got cloned for the next steps.

Installing the utility.

  1. Run ./install.sh (Warning: This will require superuser priviledges.)

Using it.

(For a quick explanation of the options, run bashutil -h)

An example of using it: bashutil -d ~/foo -f bar

In this example, -d means the directory of the new script, and -f means the filename of the new script. After running this, you will be prompted if the information is correct. After answering this, the utility will proceed.

Options:

  • -h: Shows the help screen.
  • -v: Shows the current version of the installed utility.
  • -f: Gets the filename. For examples look above.
  • -d: Gets the directory of the new script. For examples look above.
